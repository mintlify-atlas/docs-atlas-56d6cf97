## Get Overview KPIs

Retrieve high-level key performance indicators for the dashboard.

```http
GET /api/analytics/kpis
```

<Note>
  This endpoint is cached for 5 minutes to optimize performance.
</Note>

### Example Request

<CodeGroup>
```bash cURL
curl https://api.realtorspractice.com/api/analytics/kpis \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'https://api.realtorspractice.com/api/analytics/kpis',
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.realtorspractice.com/api/analytics/kpis',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
data = response.json()
```
</CodeGroup>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  KPI metrics object
  
  <ResponseField name="totalProperties" type="number" required>
    Total number of properties in the database (excluding soft-deleted)
  </ResponseField>
  
  <ResponseField name="activeProperties" type="number" required>
    Number of properties with status `AVAILABLE`
  </ResponseField>
  
  <ResponseField name="totalSites" type="number" required>
    Total number of configured scraping sites
  </ResponseField>
  
  <ResponseField name="avgQuality" type="number" required>
    Average quality score across all properties (0-100, rounded)
  </ResponseField>
  
  <ResponseField name="newToday" type="number" required>
    Number of properties added today (since midnight)
  </ResponseField>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

### Example Response

```json
{
  "success": true,
  "data": {
    "totalProperties": 12847,
    "activeProperties": 11203,
    "totalSites": 8,
    "avgQuality": 78,
    "newToday": 143
  },
  "message": "KPIs retrieved successfully"
}
```

## Get Chart Data

Retrieve aggregated data for dashboard visualizations.

```http
GET /api/analytics/charts
```

<Note>
  This endpoint is cached for 5 minutes to optimize performance.
</Note>

### Example Request

<CodeGroup>
```bash cURL
curl https://api.realtorspractice.com/api/analytics/charts \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'https://api.realtorspractice.com/api/analytics/charts',
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.realtorspractice.com/api/analytics/charts',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
data = response.json()
```
</CodeGroup>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  Chart data object
  
  <ResponseField name="byCategory" type="array" required>
    Properties grouped by listing type and category
    
    <ResponseField name="listingType" type="enum" required>
      `SALE`, `RENT`, `LEASE`, or `SHORTLET`
    </ResponseField>
    
    <ResponseField name="category" type="enum" required>
      `RESIDENTIAL`, `COMMERCIAL`, `LAND`, `SHORTLET`, or `INDUSTRIAL`
    </ResponseField>
    
    <ResponseField name="count" type="number" required>
      Number of properties matching this combination
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="byStatus" type="array" required>
    Properties grouped by status (for donut/pie chart)
    
    <ResponseField name="name" type="enum" required>
      Status name: `AVAILABLE`, `SOLD`, `RENTED`, `UNDER_OFFER`, `WITHDRAWN`, or `EXPIRED`
    </ResponseField>
    
    <ResponseField name="value" type="number" required>
      Number of properties with this status
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

## Get Site Quality Rankings

Retrieve quality rankings for all configured scraping sites.

```http
GET /api/analytics/site-quality
```

### Example Request

<CodeGroup>
```bash cURL
curl https://api.realtorspractice.com/api/analytics/site-quality \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'https://api.realtorspractice.com/api/analytics/site-quality',
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  }
);
const data = await response.json();
```
</CodeGroup>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array" required>
  Array of site quality rankings
  
  <ResponseField name="siteId" type="string" required>
    Site ID (CUID)
  </ResponseField>
  
  <ResponseField name="siteName" type="string" required>
    Human-readable site name
  </ResponseField>
  
  <ResponseField name="siteKey" type="string" required>
    Unique site key identifier
  </ResponseField>
  
  <ResponseField name="healthScore" type="number" required>
    Overall health score (0-100)
  </ResponseField>
  
  <ResponseField name="totalListings" type="number" required>
    Total number of properties from this site
  </ResponseField>
  
  <ResponseField name="avgQuality" type="number" required>
    Average quality score of properties from this site (0-100)
  </ResponseField>
  
  <ResponseField name="freshnessScore" type="number" required>
    Data freshness score based on last scrape time (0-100)
  </ResponseField>
  
  <ResponseField name="lastScrapeAt" type="string">
    ISO 8601 timestamp of last successful scrape
  </ResponseField>
  
  <ResponseField name="daysSinceLastScrape" type="number">
    Number of days since last scrape
  </ResponseField>
  
  <ResponseField name="failCount" type="number" required>
    Number of consecutive scrape failures
  </ResponseField>
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

## Usage Examples

### Dashboard Overview Widget

```javascript
import { useQuery } from '@tanstack/react-query';

function DashboardOverview() {
  const { data, isLoading } = useQuery({
    queryKey: ['analytics', 'kpis'],
    queryFn: async () => {
      const res = await fetch('https://api.realtorspractice.com/api/analytics/kpis', {
        headers: { 'Authorization': `Bearer ${token}` }
      });
      return res.json();
    },
    staleTime: 5 * 60 * 1000, // 5 minutes
  });

  if (isLoading) return <Skeleton />;

  return (
    <div className="grid grid-cols-5 gap-4">
      <StatCard title="Total Properties" value={data.data.totalProperties} />
      <StatCard title="Active" value={data.data.activeProperties} />
      <StatCard title="Sites" value={data.data.totalSites} />
      <StatCard title="Avg Quality" value={`${data.data.avgQuality}%`} />
      <StatCard title="New Today" value={data.data.newToday} trend="up" />
    </div>
  );
}
```

### Properties by Category Chart

```javascript
import { BarChart, Bar, XAxis, YAxis, Tooltip, Legend } from 'recharts';
import { useQuery } from '@tanstack/react-query';

function CategoryChart() {
  const { data } = useQuery({
    queryKey: ['analytics', 'charts'],
    queryFn: async () => {
      const res = await fetch('https://api.realtorspractice.com/api/analytics/charts', {
        headers: { 'Authorization': `Bearer ${token}` }
      });
      return res.json();
    },
    staleTime: 5 * 60 * 1000,
  });

  // Transform data for visualization
  const chartData = data?.data.byCategory.reduce((acc, item) => {
    const existing = acc.find(x => x.category === item.category);
    if (existing) {
      existing[item.listingType] = item.count;
    } else {
      acc.push({ category: item.category, [item.listingType]: item.count });
    }
    return acc;
  }, []);

  return (
    <BarChart width={600} height={300} data={chartData}>
      <XAxis dataKey="category" />
      <YAxis />
      <Tooltip />
      <Legend />
      <Bar dataKey="SALE" fill="var(--primary)" />
      <Bar dataKey="RENT" fill="var(--accent)" />
      <Bar dataKey="LEASE" fill="#0a6906" />
      <Bar dataKey="SHORTLET" fill="#8b5cf6" />
    </BarChart>
  );
}
```

### Status Distribution Donut

```javascript
import { PieChart, Pie, Cell, Tooltip, Legend } from 'recharts';
import { useQuery } from '@tanstack/react-query';

const STATUS_COLORS = {
  AVAILABLE: '#0a6906',
  SOLD: '#0001FC',
  RENTED: '#FF6600',
  UNDER_OFFER: '#f59e0b',
  WITHDRAWN: '#6b7280',
  EXPIRED: '#ef4444'
};

function StatusDonut() {
  const { data } = useQuery({
    queryKey: ['analytics', 'charts'],
    queryFn: async () => {
      const res = await fetch('https://api.realtorspractice.com/api/analytics/charts', {
        headers: { 'Authorization': `Bearer ${token}` }
      });
      return res.json();
    },
  });

  return (
    <PieChart width={400} height={300}>
      <Pie
        data={data?.data.byStatus}
        dataKey="value"
        nameKey="name"
        cx="50%"
        cy="50%"
        innerRadius={60}
        outerRadius={100}
      >
        {data?.data.byStatus.map((entry, index) => (
          <Cell key={index} fill={STATUS_COLORS[entry.name]} />
        ))}
      </Pie>
      <Tooltip />
      <Legend />
    </PieChart>
  );
}
```

## Caching

Analytics endpoints use Redis caching with a 5-minute TTL to optimize performance and reduce database load. This means:

- Data may be up to 5 minutes old
- Subsequent requests within the cache window are served instantly
- Cache is automatically invalidated after 5 minutes

<Tip>
  For real-time property counts or immediate updates after data changes, consider using the `/api/properties/count` endpoint instead.
</Tip>
