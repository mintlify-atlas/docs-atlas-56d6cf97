## Endpoint

```
GET /api/properties/:id/price-history
```

Retrieve a simplified price change history for a specific property, showing all recorded price points over time.

## Authentication

Requires authentication with any user role (Viewer, Editor, or Admin).

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier (CUID) of the property
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indicates whether the request was successful
</ResponseField>

<ResponseField name="data" type="PriceHistoryEntry[]" required>
  Array of price history records in chronological order
  
  <Expandable title="PriceHistoryEntry fields">
    <ResponseField name="id" type="string">
      Price history record ID
    </ResponseField>
    
    <ResponseField name="price" type="number">
      The recorded price value (in NGN)
    </ResponseField>
    
    <ResponseField name="source" type="string">
      Source of the price record (e.g., "scraper", "manual_edit", "initial")
    </ResponseField>
    
    <ResponseField name="recordedAt" type="string">
      ISO timestamp when the price was recorded
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://realtors-practice-new-api.onrender.com/api/properties/clx123abc/price-history" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const propertyId = 'clx123abc';

const response = await fetch(
  `https://realtors-practice-new-api.onrender.com/api/properties/${propertyId}/price-history`,
  {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  }
);

const { data } = await response.json();

// Calculate price change percentage
if (data.length >= 2) {
  const initial = data[0].price;
  const current = data[data.length - 1].price;
  const change = ((current - initial) / initial) * 100;
  console.log(`Price change: ${change.toFixed(2)}%`);
}
```
</CodeGroup>

## Use Cases

- **Price Trend Visualization**: Plot price changes on a chart
- **Price Drop Alerts**: Trigger notifications when price decreases
- **Market Analysis**: Analyze pricing patterns across properties
- **Investment Insights**: Track property value appreciation/depreciation
- **Negotiation Data**: Show price reduction history to buyers

## Chart Integration Example

```jsx
import { LineChart, Line, XAxis, YAxis, Tooltip, ResponsiveContainer } from 'recharts';

function PriceHistoryChart({ propertyId }) {
  const [history, setHistory] = useState([]);

  useEffect(() => {
    fetch(`/api/properties/${propertyId}/price-history`, {
      headers: { 'Authorization': `Bearer ${token}` }
    })
      .then(res => res.json())
      .then(({ data }) => setHistory(data));
  }, [propertyId]);

  const chartData = history.map(entry => ({
    date: new Date(entry.recordedAt).toLocaleDateString(),
    price: entry.price / 1000000 // Convert to millions
  }));

  return (
    <ResponsiveContainer width="100%" height={300}>
      <LineChart data={chartData}>
        <XAxis dataKey="date" />
        <YAxis label={{ value: 'Price (₦M)', angle: -90 }} />
        <Tooltip formatter={(value) => `₦${value}M`} />
        <Line type="monotone" dataKey="price" stroke="#0404fc" strokeWidth={2} />
      </LineChart>
    </ResponsiveContainer>
  );
}
```

## Notes

- Price history entries are returned in chronological order (oldest first)
- The first entry typically has `source: "initial"` representing the creation price
- Subsequent entries are created when:
  - The scraper detects a price change on the source site
  - An editor manually updates the price
  - The price is corrected during data enrichment
- Not all properties will have price history (single-version properties return only the initial price)
- Price values are stored in NGN (Nigerian Naira)
