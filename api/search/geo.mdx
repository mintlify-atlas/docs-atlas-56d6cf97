## Overview

Geospatial search endpoints enable map-based property discovery using coordinates. All geo endpoints require authentication.

<Note>
  All endpoints require `Authorization: Bearer <token>` header.
</Note>

---

## Bounding Box Search

### GET /api/geo/bbox

Retrieve properties within a rectangular map viewport defined by north, south, east, and west coordinates. Optimized for map marker rendering.

#### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase authentication
</ParamField>

#### Query Parameters

<ParamField query="north" type="number" required>
  Maximum latitude (-90 to 90)
</ParamField>

<ParamField query="south" type="number" required>
  Minimum latitude (-90 to 90)
</ParamField>

<ParamField query="east" type="number" required>
  Maximum longitude (-180 to 180)
</ParamField>

<ParamField query="west" type="number" required>
  Minimum longitude (-180 to 180)
</ParamField>

<ParamField query="listingType" type="string">
  Filter by listing type: SALE, RENT, LEASE, SHORTLET
</ParamField>

<ParamField query="category" type="string">
  Filter by category: RESIDENTIAL, COMMERCIAL, LAND, SHORTLET, INDUSTRIAL
</ParamField>

<ParamField query="minPrice" type="number">
  Minimum price in NGN
</ParamField>

<ParamField query="maxPrice" type="number">
  Maximum price in NGN
</ParamField>

<ParamField query="bedrooms" type="integer">
  Minimum number of bedrooms
</ParamField>

#### Response

<ResponseField name="success" type="boolean" required>
  Whether the request succeeded
</ResponseField>

<ResponseField name="data" type="array" required>

  <ResponseField name="id" type="string">

  <ResponseField name="price" type="number">
    Price in NGN
  </ResponseField>
  
  <ResponseField name="latitude" type="number">
    Property latitude
  </ResponseField>
  
  <ResponseField name="longitude" type="number">
    Property longitude
  </ResponseField>

    Neighborhood name

    Nigerian state

    SALE, RENT, LEASE, or SHORTLET

  <ResponseField name="bedrooms" type="integer">
    Number of bedrooms

```javascript JavaScript - Map Pan Handler
      lat: property.latitude,
      lng: property.longitude,
      title: property.title,
      price: property.price,
      id: property.id
    });
  });
});
```
</CodeGroup>

## Radius Search

### GET /api/geo/radius

Find properties within a specified radius of a point, sorted by distance. Uses Haversine formula for accurate distance calculation.

#### Query Parameters

<ParamField query="lat" type="number" required>
  Center latitude (-90 to 90)
</ParamField>

<ParamField query="lng" type="number" required>
  Center longitude (-180 to 180)
</ParamField>

<ParamField query="radiusKm" type="number" default="5">
  Search radius in kilometers (0.1 to 100)
</ParamField>

<ParamField query="listingType" type="string">
  Filter by listing type
</ParamField>

<ParamField query="category" type="string">
  Filter by category
</ParamField>

<ParamField query="minPrice" type="number">
  Minimum price in NGN
</ParamField>

<ParamField query="maxPrice" type="number">
  Maximum price in NGN
</ParamField>

<ParamField query="bedrooms" type="integer">
  Minimum number of bedrooms
</ParamField>

#### Response

Same structure as bounding box search, with additional `distance` field:

<ResponseField name="distance" type="number">
  Distance from center point in kilometers (rounded to 2 decimals)
</ResponseField>

#### Example Request

<CodeGroup>
```bash cURL - 2km Around Lekki Phase 1
curl -X GET "https://realtors-practice-new-api.onrender.com/api/geo/radius?lat=6.4434&lng=3.4766&radiusKm=2&listingType=RENT" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## Geocode Area

### GET /api/geo/geocode

Convert a Nigerian area name to coordinates using built-in lookup table. No external API calls required.

#### Query Parameters

<ParamField query="area" type="string" required>
  Area name (case-insensitive). Supports partial matching.
  
  Examples: "lekki", "victoria island", "vi", "maitama"
</ParamField>

#### Response

<ResponseField name="success" type="boolean" required>
  Whether the request succeeded
</ResponseField>

<ResponseField name="data" type="object">
  Area coordinates
  
  <ResponseField name="area" type="string">
    Original query string
  </ResponseField>
  
  <ResponseField name="lat" type="number">
    Latitude
  </ResponseField>
  
  <ResponseField name="lng" type="number">
    Longitude
  </ResponseField>
</ResponseField>

#### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://realtors-practice-new-api.onrender.com/api/geo/geocode?area=lekki+phase+1" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```javascript JavaScript - Search Bar Integration
// User types area name, geocode it to center map
async function searchArea(areaName) {
  const response = await fetch(
    'https://realtors-practice-new-api.onrender.com/api/geo/geocode?' +
    new URLSearchParams({ area: areaName }),
    {
      headers: {
        'Authorization': `Bearer ${accessToken}`
      }
    }
  );
  
  const data = await response.json();
  
  if (data.success) {
    // Center map on area
    map.setCenter({ lat: data.data.lat, lng: data.data.lng });
    map.setZoom(14);
  }
}
```
</CodeGroup>

#### Example Response

```json
{
  "success": true,
  "data": {
    "area": "lekki phase 1",
    "lat": 6.4434,
    "lng": 3.4766
  }
}
```

**Other Cities:** Ibadan (Bodija), Enugu (Independence Layout)

View full list: `GET /api/geo/areas`

  Whether the request succeeded
</ResponseField>

<ResponseField name="data" type="array" required>
  Array of area name strings (sorted A-Z)

#### Example Request

<CodeGroup>
```bash cURL

  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```javascript JavaScript - Autocomplete
const response = await fetch(
  'https://realtors-practice-new-api.onrender.com/api/geo/areas',
  {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  }
);

const data = await response.json();
const areas = data.data;

// Filter for autocomplete
function getAreaSuggestions(input) {
  const lower = input.toLowerCase();
  return areas.filter(area => area.toLowerCase().includes(lower));
}

```

## Map Integration Guide

### 1. Initial Load

```javascript
// Center map on user's selected area or default to Lagos
const defaultCenter = { lat: 6.5244, lng: 3.3792 }; // Lagos
map.setCenter(defaultCenter);
map.setZoom(11);

// Load initial properties
const bounds = map.getBounds();
const properties = await fetchBoundingBox(bounds);
renderMarkers(properties);
```

### 2. Pan/Zoom Updates

```javascript
let debounceTimer;

map.on('moveend', () => {
  clearTimeout(debounceTimer);
  
  // Debounce to avoid excessive API calls
  debounceTimer = setTimeout(async () => {
    const bounds = map.getBounds();
    const properties = await fetchBoundingBox(bounds);
    renderMarkers(properties);
  }, 300);
});
```

### 3. Clustering for Dense Areas

When >100 properties are in viewport, implement client-side clustering:

```javascript
import { MarkerClusterer } from '@googlemaps/markerclusterer';

const clusterer = new MarkerClusterer({ map, markers });
```

Alternatively, use Mapbox's built-in clustering:

```javascript
map.addSource('properties', {
  type: 'geojson',
  data: geojson,
  cluster: true,
  clusterMaxZoom: 14,
  clusterRadius: 50
});
```

### 4. Performance Optimization

- **Cache viewport queries:** Store last fetched bounds to avoid duplicate requests
- **Limit marker count:** Cap at 500 markers (already enforced server-side)
- **Use vector tiles:** For >10k properties, switch to vector tile rendering
- **Lazy load images:** Only load property images on marker click

## Nigeria Coordinate Bounds

Coordinates must be within Nigeria's geographic boundaries:

- **North:** 13.9°N
- **South:** 4.0°N
- **East:** 14.7°E
- **West:** 2.7°E

Requests with coordinates outside Nigeria will return `400 Bad Request`:

```json
{
  "success": false,
  "error": "Coordinates must be within Nigeria"
}
```

## Distance Calculation

Radius search uses the **Haversine formula** for accurate great-circle distance:

```
a = sin²(Δlat/2) + cos(lat1) × cos(lat2) × sin²(Δlng/2)
c = 2 × atan2(√a, √(1−a))
distance = R × c
```

Where `R = 6371 km` (Earth's radius)

Accuracy: within 0.5% for distances under 100km

## Notes

- All coordinates are in **WGS84 format** (standard GPS coordinates)
- Properties without `latitude` or `longitude` are excluded from geo queries
- Maximum 500 properties returned per bounding box query (sorted by quality score)
- Radius search has no hard limit but uses bounding box filter first for performance
- Geocoding is offline (no external API dependencies)
- All geo endpoints require authentication to prevent abuse
