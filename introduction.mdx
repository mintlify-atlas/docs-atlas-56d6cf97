## Platform Highlights

<AccordionGroup>
  <Accordion title="Comprehensive Property Database" icon="database">
    Store and manage properties with **85+ data fields** across 9 categories:
    - Basic information (title, source, status, category)
    - Property details (bedrooms, bathrooms, size, condition)
    - Financial data (price, fees, payment plans)
    - Location (address, coordinates, landmarks)
    - Amenities and features
    - Media (images, videos, virtual tours)
    - Agent information
    - Quality scores and metadata
    - Promotional tags
  </Accordion>

  <Accordion title="Advanced Search & Filtering" icon="filter">
    - **Meilisearch** integration for fast full-text search
    - Advanced filters: location, price range, bedrooms, property type
    - Geospatial bounding-box queries
    - Saved searches with automatic monitoring
    - Natural language query support
  </Accordion>

  <Accordion title="Version Tracking & Audit" icon="clock-rotate-left">
    - Full property version history with change diffs
    - Audit logs for all user actions
    - Price history tracking
    - Multiple change sources (scraper, manual, enrichment, system)
  </Accordion>

  <Accordion title="Role-Based Access Control" icon="users">
    - **Admin**: Full platform control and user management
    - **Editor**: Create and modify properties
    - **Viewer**: Read-only access to listings
    - **API User**: Programmatic access via API keys
  </Accordion>
</AccordionGroup>

## Tech Stack

The platform is built as a microservice architecture:

- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS v4, shadcn/ui
- **Backend**: Node.js, Express, TypeScript, Prisma ORM, Socket.io
- **Scraper**: Python 3.11, Playwright, BeautifulSoup, Flask
- **Database**: CockroachDB (PostgreSQL-compatible)
- **Search**: Meilisearch (self-hosted)
- **Auth**: Supabase (JWT authentication)
- **Deployment**: Vercel (frontend), Render (backend + scraper)

<Note>
  Realtors' Practice is designed for the Nigerian real estate market with support for local property types, pricing in NGN, and region-specific data fields.
</Note>
