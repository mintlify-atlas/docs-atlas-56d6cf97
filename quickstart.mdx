## Welcome to Realtors' Practice

Realtors' Practice is an AI-powered Nigerian property intelligence platform that aggregates, validates, and enriches real estate listings from across Nigeria. This guide will walk you through your first property search.

<Note>
  **Before you begin:** You'll need an invitation code to create an account. Contact your administrator if you don't have one.
</Note>

## Get Started in 3 Steps

<Steps>
  <Step title="Create Your Account">
    Navigate to the registration page and enter your invitation code.
    
    <Info>
      **Invitation-only access:** Realtors' Practice uses invitation codes to ensure quality and security. Each code is linked to a specific email address and role.
    </Info>

    After registration, you'll be automatically authenticated via Supabase Auth and assigned a role:
    - **VIEWER** - Browse and search properties
    - **EDITOR** - Edit property details and manage data
    - **ADMIN** - Full platform access including scraper configuration
  </Step>

  <Step title="Search for Properties">
    Once logged in, you'll land on the dashboard. Head to the **Properties** page to start searching.

    ### Search Methods

    **Natural Language Search**
    
    Type queries in plain English using the search bar:
    
    ```
    3 bedroom flat in Lekki under 5M
    Detached duplex in Ikoyi with pool
    2 bed apartment in Abuja under 3M
    Land for sale in Ajah below 20M
    ```

    The platform uses Meilisearch to instantly return relevant results based on location, price, bedrooms, and features.

    **Advanced Filters**
    
    Click the filters icon to refine by:
    - **Listing Type:** Sale, Rent, Lease, Shortlet
    - **Category:** Residential, Commercial, Land, Industrial
    - **Price Range:** Custom min/max with NGN support
    - **Bedrooms/Bathrooms:** Exact counts or ranges
    - **Location:** State, LGA, Area
    - **Features:** Pool, Security, Parking, BQ, etc.
    - **Quality Score:** Filter by data completeness (0-100%)

    **Map View**
    
    Toggle the map to visualize properties geographically. Click markers to see property details. Map providers include:
    - OpenStreetMap (default, free)
    - Mapbox
    - Google Maps
  </Step>

  <Step title="Save Searches & Get Alerts">
    Found a search you want to monitor? Save it for automatic notifications.

    ### Creating a Saved Search

    1. Apply your desired filters
    2. Click **Save Search** in the properties toolbar
    3. Name your search (e.g., "Lekki 3BR under 5M")
    4. Choose notification preferences:
       - **In-app notifications** - See new matches in your notifications panel
       - **Email alerts** - Receive daily digests of new listings

    ### How It Works

    ```json
    {
      "name": "Lekki 3BR under 5M",
      "filters": {
        "area": "Lekki",
        "bedrooms": 3,
        "priceMax": 5000000,
        "listingType": "SALE"
      },
      "notifyEmail": true,
      "notifyInApp": true
    }
    ```

    The backend runs daily cron jobs that:
    1. Check for new properties matching your saved search criteria
    2. Create `SavedSearchMatch` records for qualifying properties
    3. Send notifications via email (Resend) and in-app (Socket.io)

    <Info>
      New matches are checked against your saved searches automatically. You'll receive notifications for price drops and new listings that match your criteria.
    </Info>
  </Step>
</Steps>

## Understanding Property Data

Each property in the platform has rich, structured data:

### Property Structure

```typescript
interface Property {
  // Core Info
  id: string;                    // Unique identifier (cuid)
  title: string;                 // "4 Bedroom Detached Duplex"
  listingType: ListingType;      // SALE | RENT | LEASE | SHORTLET
  category: PropertyCategory;    // RESIDENTIAL | COMMERCIAL | LAND | INDUSTRIAL
  status: PropertyStatus;        // AVAILABLE | SOLD | RENTED | UNDER_OFFER
  
  // Pricing
  price: number;                 // Amount in NGN
  priceNegotiable: boolean;
  rentFrequency?: string;        // "annually" | "monthly"
  serviceCharge?: number;
  
  // Property Details
  bedrooms: number;
  bathrooms: number;
  toilets: number;
  bq: number;                    // Boy's quarters
  landSizeSqm: number;          // Land size in square meters
  buildingSizeSqm: number;      // Building size in square meters
  furnishing: Furnishing;        // FURNISHED | SEMI_FURNISHED | UNFURNISHED
  
  // Location
  state: string;                 // "Lagos"
  lga?: string;                  // Local Government Area
  area?: string;                 // "Lekki Phase 1"
  estateName?: string;           // "Chevron Estate"
  latitude?: number;
  longitude?: number;
  
  // Features & Amenities
  features: string[];            // ["Swimming Pool", "Gym", "24hr Security"]
  security: string[];
  utilities: string[];
  parkingSpaces: number;
  
  // Media
  images: string[];              // Array of image URLs
  videos?: string[];
  virtualTourUrl?: string;
  
  // Agent Info
  agentName?: string;
  agentPhone?: string;
  agentEmail?: string;
  agencyName?: string;
  
  // Metadata
  source: string;                // "PropertyPro.ng"
  qualityScore: number;          // 0-100 based on data completeness
  verificationStatus: VerificationStatus; // UNVERIFIED | VERIFIED | FLAGGED
  daysOnMarket: number;
  viewCount: number;
}
```

### Data Quality Scoring

Properties are scored 0-100 based on data completeness:

- **Images** (25 points) - Has at least 4 quality images
- **Description** (15 points) - Detailed, informative description
- **Location** (20 points) - Full address with coordinates
- **Pricing** (15 points) - Clear pricing with breakdown
- **Features** (10 points) - Comprehensive amenities list
- **Agent Info** (10 points) - Verified contact details
- **Media** (5 points) - Virtual tour or video

Filter by `qualityScore` to see only high-quality listings.

## Viewing Property Details

Click any property card to open the detail panel. You'll see:

- **Image gallery** with full-screen viewer
- **Complete property specifications**
- **Location map** with nearby landmarks
- **Price history chart** (if price has changed)
- **Agent contact information**
- **Similar properties** based on location and price
