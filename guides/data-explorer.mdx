## Overview

The Data Explorer provides powerful analytics and visualization tools to understand the Nigerian property market. Analyze pricing trends, inventory distribution, market activity, and more using data from thousands of scraped listings.

<Note>
  The Data Explorer is available to all user roles. Admins and Editors can access all features, while Viewers have read-only access to visualizations.
</Note>

## Price Analysis

Deep dive into pricing data to understand market rates and identify opportunities.

### Price Distribution

Visualize how properties are priced across different segments:

<Steps>
  <Step title="Select filters">
    Choose:
    - Location (state, area)
    - Property category and type
    - Bedroom count
    - Listing type (sale vs. rent)
  </Step>
  
  <Step title="View histogram">
    See the distribution of prices in your selected segment. Identify:
    - Price clusters (most common price ranges)
    - Outliers (unusually high/low prices)
    - Market gaps (price ranges with few listings)
  </Step>
  
  <Step title="Calculate statistics">
    Review:
    - Mean, median, mode prices
    - Standard deviation
    - Min/max values
    - Quartiles and percentiles
  </Step>
</Steps>

### Price Per SQM Analysis

Compare value across properties of different sizes:


## Export & API Access

Access analytics data programmatically for custom analysis.

### Export Data

Export filtered property datasets:

<CodeGroup>
```bash CSV Export
curl -X GET "https://api.realtorspractice.com/api/properties/export" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  --data-urlencode "format=csv" \
  --data-urlencode "filters=state=Lagos AND bedrooms=3" \
  -o properties.csv
```

```bash JSON Export
curl -X GET "https://api.realtorspractice.com/api/properties/export" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  --data-urlencode "format=json" \
  --data-urlencode "filters=category=RESIDENTIAL" \
  -o properties.json
```
</CodeGroup>

### Analytics Endpoints

Query aggregated statistics via API:

```bash Price Statistics
curl -X GET "https://api.realtorspractice.com/api/analytics/price-stats" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  --data-urlencode "area=Lekki Phase 1" \
  --data-urlencode "bedrooms=3"
```

## Common Analysis Scenarios

<AccordionGroup>
  <Accordion title="Investment Area Selection">
    Compare areas by:
    1. Average price per sqm (affordability)
    2. Days on market (liquidity)
    3. Listing volume growth (demand)
    4. Rental yield (for buy-to-let)
    
    Look for areas with improving metrics but still below premium pricing.
  </Accordion>
  
  <Accordion title="Portfolio Valuation">
    Use similar property pricing to:
    1. Filter by exact property characteristics
    2. Calculate price per sqm for comparables
    3. Adjust for quality score differences
    4. Account for location premiums
    
    Average 3-5 comparables for reliable valuation.
  </Accordion>
  
  <Accordion title="Market Timing">
    Analyze:
    1. New listing velocity (supply)
    2. Days on market trend (demand)
    3. Price trajectory (direction)
    4. Seasonal patterns (timing)
    
    Rising prices + falling DOM + increasing supply = hot market.
  </Accordion>
  
  <Accordion title="Feature Optimization">
    For developers, identify:
    1. Most demanded features in target area
    2. Features with highest price premium
    3. Underserved feature combinations
    4. Cost vs. value analysis
  </Accordion>
</AccordionGroup>

## Limitations & Considerations

<Warning>
  **Data Accuracy Notes:**
  - Scraped data may contain errors from source websites
  - Not all listings are verified or still available
  - Prices may not reflect actual transaction values
  - Data freshness varies by source and scraping frequency
  
  Always verify critical information directly with agents before making decisions.
</Warning>

<Note>
  **Statistical Caveats:**
  - Small sample sizes (< 20 properties) may not be statistically significant
  - Outliers can skew averages - check median values too
  - Seasonal variations affect comparisons across time periods
  - Not all properties are captured (off-market deals, exclusive listings)
</Note>
