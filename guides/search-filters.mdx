## Overview

Realtors' Practice offers powerful search capabilities powered by **Meilisearch**, combining traditional filters with natural language search. Find properties across Nigeria with precision and speed.

## Quick Search

The fastest way to find properties is using the main search bar.

<Steps>
  <Step title="Enter your search query">
    Type keywords, property types, locations, or natural language queries:
    - "3 bedroom flat in Lekki"
    - "Commercial property Victoria Island"
    - "Land for sale Ikoyi under 50 million"
  </Step>
  
  <Step title="Review results">
    Results appear instantly with highlighted matches. The search indexes:
    - Property titles
    - Descriptions
    - Location details (area, state, landmarks)
    - Property features
    - Agent information
  </Step>
</Steps>

```bash Request
curl -X GET "https://api.realtorspractice.com/api/search?q=3+bedroom+lekki&limit=20" \
```

## Advanced Filters

Refine your search with multiple filters for precise results.

### Available Filters

<AccordionGroup>
  <Accordion title="Listing Type" icon="tag">
    - **SALE**: Properties for purchase
    - **RENT**: Long-term rentals
    - **LEASE**: Lease agreements
    - **SHORTLET**: Short-term rentals
    
    Filter parameter: `filters=listingType=SALE`
  </Accordion>
  
  <Accordion title="Property Category" icon="building">
    - **RESIDENTIAL**: Homes and apartments
    - **COMMERCIAL**: Offices, shops, warehouses
    - **LAND**: Plots and land parcels
    - **SHORTLET**: Furnished short-stays
    - **INDUSTRIAL**: Factories and industrial spaces
    
    Filter parameter: `filters=category=RESIDENTIAL`
  </Accordion>
  
  <Accordion title="Bedrooms" icon="bed">
    Filter by exact number of bedrooms (1-10+).
    
    Filter parameter: `filters=bedrooms=3`
  </Accordion>
  
  <Accordion title="Price Range" icon="naira-sign">
    Set minimum and maximum price in Nigerian Naira.
    
    Filter parameters:
    - Minimum: `filters=price>=50000000`
    - Maximum: `filters=price<=100000000`
    - Combined: `filters=price 50000000 TO 100000000`
  </Accordion>
  
  <Accordion title="Location" icon="map-pin">
    - **State**: Filter by Nigerian state (e.g., Lagos, Abuja)
    - **Area**: Specific neighborhoods or districts
    
    Filter parameters:
    - State: `filters=state="Lagos"`
    - Area: `filters=area="Lekki Phase 1"`
  </Accordion>
  
  <Accordion title="Property Type" icon="house">
    Specific property subtypes like:
    - Flat / Apartment
    - Detached Duplex
    - Semi-Detached
    - Terrace
    - Bungalow
    - Mansion
    - Office Space
    - Shop
    - Warehouse
    
    Filter parameter: `filters=propertyType="Detached Duplex"`
  </Accordion>
</AccordionGroup>

### API Example: Multi-Filter Search

Combine multiple filters for targeted results.

```bash Request
curl -X GET "https://api.realtorspractice.com/api/search" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  --data-urlencode "q=luxury apartment" \
  --data-urlencode "filters=listingType=SALE AND bedrooms=3 AND state=Lagos AND price 70000000 TO 120000000" \
  --data-urlencode "limit=20" \
  --data-urlencode "offset=0"
```

```json Response
{

  "data": {
    "hits": [
```

<Note>
  **Filter Syntax:**
  - Use `AND` to combine filters: `state=Lagos AND bedrooms=3`
  - Use `OR` for alternatives: `listingType=SALE OR listingType=RENT`
  - Use `TO` for ranges: `price 50000000 TO 100000000`
  - Use quotes for exact matches: `area="Lekki Phase 1"`
</Note>

## Faceted Search

Facets provide aggregate counts of matching results by category, helping you understand the available inventory.

### Default Facets

By default, the API returns counts for:
- `categoryName` (RESIDENTIAL, COMMERCIAL, LAND, etc.)
- `listingType` (SALE, RENT, LEASE, SHORTLET)
- `bedrooms` (1, 2, 3, 4, 5+)
- `state` (Lagos, Abuja, Port Harcourt, etc.)

### Custom Facets

Request specific facets to get targeted statistics.

```bash Request
curl -X GET "https://api.realtorspractice.com/api/search" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  --data-urlencode "q=apartment lekki" \
  --data-urlencode "facets=area,propertyType,bedrooms,price"
```

```json Response
{

  "data": {
```

## Sorting Results

Control the order of search results with sort parameters.

### Available Sort Options

<CardGroup cols={2}>
  <Card title="Relevance" icon="star">
    Default sorting by search relevance score.
    
    `sort=_relevance:desc`
  </Card>
  <Card title="Date Added" icon="clock">
    Newest or oldest listings first.
    
    `sort=createdAt:desc` or `sort=createdAt:asc`
  </Card>
  <Card title="Price" icon="naira-sign">
    Highest to lowest or lowest to highest.
    
    `sort=price:desc` or `sort=price:asc`
  </Card>
  <Card title="Quality Score" icon="certificate">
    Best quality listings first.
    
    `sort=qualityScore:desc`
  </Card>
</CardGroup>

### Multi-Field Sorting

Combine multiple sort criteria for refined ordering.

```bash Request
curl -X GET "https://api.realtorspractice.com/api/search" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  --data-urlencode "q=flat lagos" \
  --data-urlencode "sort=qualityScore:desc,price:asc"
```

<Note>
  Results are sorted by the first criterion, then ties are broken by subsequent criteria.
</Note>

## Pagination

Navigate through large result sets efficiently.

### Parameters
- `limit`: Number of results per page (default: 20, max: 100)
- `offset`: Starting position (default: 0)

```bash Request
curl -X GET "https://api.realtorspractice.com/api/search?q=apartment&limit=20&offset=40" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

<Tip>
  **Pagination Tips:**
  - For page 1: `offset=0, limit=20`
  - For page 2: `offset=20, limit=20`
  - For page 3: `offset=40, limit=20`
  
  Formula: `offset = (page - 1) × limit`
</Tip>
